# 🧠 PRAIME-AI

**Praime-AI** is an AI-powered web application built with Flask, Docker, and PostgreSQL. It features user authentication (including Google OAuth), secure data storage, and a modular backend ideal for integrating intelligent features like chat, analysis, or ML models.

---

## 🚀 Features

- ✅ User authentication with Google OAuth
- ✅ PostgreSQL database integration
- ✅ Containerized with Docker and Docker Compose
- ✅ RESTful Flask API ready for AI extensions
- ✅ Scalable and production-ready structure

---

## 🛠️ Tech Stack

- **Backend**: Flask, SQLAlchemy
- **Database**: PostgreSQL
- **Authentication**: Google OAuth

- **Containerization**: Docker, Docker Compose
- **ORM**: SQLAlchemy
